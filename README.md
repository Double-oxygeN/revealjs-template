# Reveal.js Template

This is a template repository for Reveal.js + Vite presentations.
It also includes useful Agent Skills for GitHub Copilot.

## How to Use Template

### From GitHub

1. Click "Use this template" button on GitHub and create a new repository.
1. Clone the new repository to your local machine.
1. Run `npm ci` to install dependencies.

### From GitHub CLI

1. `gh repo create <new-repo-name> -p Double-oxygeN/revealjs-template --clone`
1. `cd <new-repo-name>`
1. Run `npm ci` to install dependencies.

## Development

Run `npm run dev` to start the development server.
You can access the presentation at `http://localhost:5173`.
